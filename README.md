## Conteúdo Abordado

> **Minhas anotações e impressões sobre o Bootcamp de Lógica de Programação com JavaScript e JSON (DIO & BLIP)**  
> _Por João Pedro Moura Santos_

---

### 1. Introdução à Lógica de Programação

Ao assistir as aulas iniciais, entendi que a lógica de programação é o ponto de partida para qualquer pessoa que deseja aprender a programar. O instrutor explicou que pensar logicamente é essencial para criar soluções eficientes e que, antes mesmo de aprender uma linguagem específica, é importante compreender como estruturar o raciocínio para resolver problemas.

---

### 2. Algoritmos e Fluxogramas

Aprendi que algoritmos são sequências de passos bem definidos para chegar a uma solução. Achei interessante a abordagem dos fluxogramas, pois eles ajudam muito a visualizar o fluxo do programa e entender a ordem das operações, facilitando tanto o planejamento quanto a comunicação de ideias.

---

### 3. Tipos de Dados e Variáveis em JavaScript

Nas aulas sobre variáveis, ficou claro que o JavaScript trabalha com diferentes tipos de dados, como números, textos (strings) e valores booleanos (verdadeiro ou falso). Também vi como arrays e objetos são importantes para organizar e estruturar conjuntos de informações, e como a linguagem é flexível por ter tipagem dinâmica.

---

### 4. Operadores Lógicos e Aritméticos

Os operadores aritméticos (como +, -, *, /) e lógicos (&&, ||, !) são ferramentas básicas, mas essenciais. Eles permitem fazer cálculos, comparar valores e tomar decisões dentro do código. Ver exemplos práticos ajudou bastante a fixar como e quando usar cada um.

---

### 5. Estruturas de Controle: Decisão e Repetição

Achei fundamental entender as estruturas de controle, como `if`, `else`, `switch` para decisões, e `for`, `while`, `do...while` para repetições. Essas estruturas tornam o código muito mais dinâmico, permitindo executar diferentes ações de acordo com condições específicas ou repetir tarefas automaticamente.

---

### 6. Funções e Modularização

Vi que funções são blocos de código reutilizáveis, ótimos para organizar melhor o programa e evitar repetição. Aprendi sobre parâmetros, retorno de valores e como o escopo de variáveis influencia no funcionamento das funções. Isso tudo facilita bastante a manutenção do código.

---

### 7. Manipulação de Arrays e Objetos

As aulas mostraram como arrays e objetos são fundamentais para lidar com coleções de dados em JavaScript. Pratiquei métodos para adicionar, remover, acessar e modificar elementos, o que é indispensável para trabalhar com informações em projetos reais.

---

### 8. Introdução ao JSON: Estrutura, Leitura e Escrita

Gostei de aprender sobre JSON, que é um formato muito usado para troca de dados entre sistemas. Vi como converter objetos JavaScript em JSON (`JSON.stringify()`) e o contrário (`JSON.parse()`), o que é muito útil para integrar aplicações e manipular dados de APIs.

---

### 9. Prática com Exercícios e Mini-Projetos

Por fim, os exercícios e mini-projetos ajudaram a fixar todo o conteúdo. Mesmo só assistindo, percebi como é importante praticar para consolidar o aprendizado, resolver desafios de lógica e aplicar os conceitos de forma mais concreta.

---

Essas anotações refletem o que absorvi ao acompanhar o bootcamp. Recomendo o estudo desses tópicos para quem está começando, pois são a base para evoluir em programação e em outras tecnologias.
